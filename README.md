# POO - Programación Orientada a Objetos

## Definición de Clase
Una **clase** es un molde o plantilla que define la estructura y comportamiento de los objetos. Especifica qué atributos (propiedades) y métodos (acciones) tendrán los objetos creados a partir de ella.

## Definición de Objeto
Un **objeto** es una instancia específica de una clase. Es un ejemplo concreto creado a partir del molde que define la clase, con valores particulares para sus atributos.

## Clase: Puerta

**Atributos:**
- `color` - El color de la puerta
- `tamaño` - El tamaño de la puerta

**Métodos:**
- `abrir(dia, hora)` - Abre la puerta registrando el día y hora de apertura
- `cerrar()` - Cierra la puerta y muestra que está cerrada con la hora en que estuvo abierta

## Ejemplo de Objeto
`puerta1` es un objeto (instancia) de la clase Puerta con color específico y tamaño definido.
